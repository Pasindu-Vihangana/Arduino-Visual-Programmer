## About
Educational digital logic simulator.

Achieves fast simulation using LLVM JIT.

Thanks to Logisim for inspiration.

## Dependencies
- Python 3
- PySide 2
- llvmlite
- sortedcontainers


## Running
1. install dependencies: `pip install -r requirements.txt`
2. run: `python main.py`
